# MOOC frontend

## Setup

1. Node.js v22
2. Install dependenices

   ```
    npm install
   ```

3. Run local development server

   ```
   npm run dev
   ```

   Open [localhost:5173](http://localhost:5173/) in browser

## Architecture

[FSD (Feature-Sliced Design)](https://feature-sliced.github.io/documentation/) architecture is used in the project. Please stick to it
